# Python Data Structures Exercises (Listas e Dicionários)

Este repositório contém a resolução de um conjunto de atividades práticas em Python focadas na manipulação de estruturas de dados fundamentais: listas (lists) e dicionários (dictionaries).

## Atividades Incluídas:
1.  **Filmes Favoritos:** Uso de `list` e `enumerate()` com laço `for`.
2.  **Material Escolar:** Manipulação de dois dicionários e uso da função `update()` com laço `while`.
3.  **Cadastro de Alunos:** Implementação de um sistema simples de cadastro utilizando dicionários aninhados, com a matrícula como chave principal.
